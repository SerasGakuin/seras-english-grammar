# seras-english-grammar

セラス学院で使用する英語参考書をデジタルテキスト化するプロジェクトです。

## このプロジェクトについて

紙の参考書（PDF）を、検索・編集可能なテキストデータ（Markdown形式）に変換しています。

### 対象書籍

| 書籍名 | 章数 | 進捗 |
|--------|------|------|
| 核心 | 21章 | 完了 |
| 成川 | 162章 | 完了 |
| 成川 別冊 | 16章 | 未着手 |
| スクランブル | 32章 | 未着手 |
| 肘井 | 38章 | 未着手 |
| 肘井 別冊 | 5章 | 未着手 |
| 入門英文 | 72章 | 未着手 |
| 入門英文 別冊 | 36章 | 未着手 |
| はじめの英文読解ドリル | 35章 | 未着手 |

## フォルダ構成

```
pdf/
├── raw/          # 元のPDFファイル
├── rotated/      # 回転済みPDF
├── images/       # 画像化したページ
└── output/       # 完成したMarkdownファイル ← ここが成果物

progress/
└── status.json   # 進捗管理ファイル
```

## 成果物の場所

変換済みのテキストは `pdf/output/` フォルダにあります。

```
pdf/output/
├── 核心/
│   ├── 00_目次.md
│   ├── Part1_01_冠詞.md
│   ├── Part1_02_代名詞.md
│   └── ...
└── 成川/
    ├── 00_目次.md
    ├── Part01_01_現在形.md
    └── ...
```
