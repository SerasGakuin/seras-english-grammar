## Theme 72　no / not の用法

🔊072

### 問題 264
〈頻出〉

**問題文**
It is obvious that (　　　) fly.

① any dog can't　② not any dog can　③ none of dog can　④ no dog can

（関西外語大）

**解説タイトル**: no + 名詞「どの〜（名詞）も…ない」（= not ... any + 名詞）―― 全体否定

**解説**
▶ no が修飾する名詞は単数・複数どちらも可。本問は単数名詞 dog を修飾。
▶「no + 名詞」は「not ... any + 名詞」で言い換えられる。
　I have *no* sisters. = I *don't* have *any* sisters.（姉妹はいません）
〈選択肢〉「any + 名詞 ... not」という語順はないので、① any dog can't は不可。「not any + 名詞」は「no + 名詞」と異なり主語にできないので ② not any dog can は不可。③ は none of the dogs can とすれば可。

**解答**: ④

**日本語訳**: どの犬も空を飛べない［空を飛べる犬がいない］のは明らかである。

### 問題 265

**問題文**
あなたの親切な提案がなければ、彼はこの計画を実行できなかっただろうと私は思います。

Without your kind suggestions, I (carried / could / don't / have / he / think) this project.

（武庫川女子大）

**解説タイトル**: I don't think 〜「〜ではないと思う」―― not の繰り上げ

**解説**
▶ 英語では、文意が変わらないなら、否定語をなるべく前に置く傾向がある。「〜ではないと思う」と言うとき、I think 〜 not ... とし ないで、I don't think 〜とするほうがふつう。I think he couldn't have ... の形はまれ。
〈発展〉I don't think 型の動詞は他に believe / expect / guess / suppose などがある。
　I **don't suppose** he will come.（彼は来ないと思うよ）
　*cf*. I hope he won't come.（彼が来なければいいな）
　＊ hope は常に肯定的な意味を持つので、not の繰り上げは行われない。

**解答**: I (don't think he could have carried out) this project.

**日本語訳**: Without your kind suggestions, I (don't think he could have carried out) this project.

## Theme 73　弱い否定

🔊073

hardly / seldom / few などは「ほとんど［めったに］〜ない」という意味で弱い否定を表す。これらは準否定語と呼ばれ、次の3つに分けられる。
(1) □ hardly / scarcely「ほとんど〜ない」―― 「程度」（➡ 266）
(2) □ seldom / rarely　「めったに〜ない」―― 「頻度」（➡ 267）
(3) □ few / little　　　「ほとんど〜ない」―― 「数量」（➡ 639・640）

---

**解説**

### Power Up! 49　強い否定の意味を表す no 〜
〈発展〉

## Theme 74　部分否定

🔊074

「すべて〜でない」のように全体を否定する用法を全体否定という。それに対し、「すべてが〜というわけではない」のように部分的に否定する用法を部分否定という。all / every / both / always / altogether などの語が否定語とともに用いられると、「〜というわけではない」という部分否定になる。否定語は all [every, etc.] より前に置くのがふつう。

---

### 問題 270
〈頻出〉

**問題文**
Some of us can dance well, but some of us can't.
= Not (　　　) of us can dance well.

（大学名なし）

**解説タイトル**: not (〜) all「すべてが〜というわけではない」

**解説**
▶ Some 〜 some [others] ...「〜のものもいれば、…のものもいる」（➡ 314）
　= Not all 〜「全員が〜というわけではない」と考える。
〈比較〉全体否定は、None of us can dance well.（私たちの誰もうまく踊れない）。
　every には形容詞の用法しかないので、not every の後ろには必ず単数名詞がくる。
　本問 = **Not every** *one* of us can dance well.

**解答**: all

**日本語訳**: 私たちの中にはうまく踊れる人もいるし、そうでない人もいる。
　私たち全員がうまく踊れるわけではない。

### 問題 271

**問題文**
I didn't read both of the books.
= I (　　　) only (　　　) of the books.

（大学名なし）

**解説タイトル**: not (〜) both「両方とも〜というわけではない」

**解説**
▶ not 〜 both「両方とも〜というわけではない」=「1つは (one) 〜」と考え、(　　　) を埋める。
〈比較〉全体否定は、I didn't read **either** of the books. = I read **neither** of the books.（私はその本のどちらも読まなかった）。
　＊ neither は2者、none は3者以上に用いることも押さえておこう。

**解答**: read, one

**日本語訳**: 私はその本の両方を読んだわけではない。
　私はその本の1冊しか読まなかった。

### Power Up! 52　強い否定を表す副詞句
〈発展〉

□ **by no means**「（どんな手段によっても）決して〜でない」（➡ 275）
□ **on no account**「（どんな理由があっても）決して〜でない」（➡ 276）
□ **in no way**「（どんな点でも）決して［少しも］〜でない」
□ **under no circumstances**「（どんな事情があっても）決して〜でない」
＊上記の表現は否定の副詞句なので、文頭に出ると後は倒置形（疑問文の語順）になる。
　**In no way** *is this* your fault.（これがあなたのせいだなんてことは決してない）
　**Under no circumstances** *should you* give up.（決してあきらめてはならない）

### Power Up! 53　その他の二重否定表現

二重否定には、他に次のような表現がある。
① It **never** rains **but**\* it pours. = It **never** rains **without** pouring.
　（降れば必どしゃ降り［2度あることは3度ある］）（ことわざ）
　\*but は「〜しないでは（…ない）」という意味の接続詞。
② **Not** a week passed **without** some unexpected problem arising.
　（毎週必ず予期せぬ問題が起こった）
③ There is **no** rose **without** a thorn.

④ He **never fails to** show up on time.（➡ Power Up! 98）

　（この砂漠で冬の間に雨が降るのはごくふつうのことだ）

### Power Up! 54　far from 〜 と free from 〜 の使い分け

(1) A is **far from** B = A **is not** B **at all**（SVC が隠れている！）
　This report is **far from** satisfactory. = This <u>report</u> <u>is</u> <u>not</u> <u>satisfactory</u> at all.
　（この報告書は決して十分ではない）　　　　　S　　V　　　　　　　C
(2) A is **free from** B = A **does not have** B（SVO が隠れている！）
　He is **free from** any fear. = He does not have any fear. = <u>He</u> <u>has</u> <u>no fear</u>.
　（彼には何の不安もない）　　　　　　　　　　　　　　　　 S　 V　　　O

### Power Up! 55　remain to be *done* と have [be] yet to *do* [be *done*]
〈発展〉

**remain to be** ***done*** = be yet to be *done* = have yet to be *done* の関係。
　The problem **remains to be** *solved*. = The problem **is** yet **to be** *solved*.
　= The problem **has** yet **to** *solve* the problem. = We **have** yet **to** *solve* the problem.
