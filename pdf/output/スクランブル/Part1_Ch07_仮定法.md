# 第7章　仮定法

## Theme 46　仮定法過去

🔊046

**基本形** If + **S** + 過去形〜, **S'** + 助動詞の過去形 + 原形 ...

「もし（今）〜なら，…だろう（に）」と，現在の事実に反することや可能性の低いことを仮定・想像する構文。過去形の動詞を使うので，『仮定法過去』と呼ばれるが，表している内容は現在のことである。

---

### 問題 158
〈頻出〉

**問題文**
If I (　　) in your situation, I would be more careful about what you post on social media.

① were　② am　③ will be　④ would be

（南山大）

**解説タイトル**: 仮定法過去 —— if 節の形

**解説**
▶ 主節の would be に注目して，仮定法過去だと判断する。仮定法過去の if 節は過去形なので，① were を選ぶ。

▶ if 節中の be 動詞は，主語の人称や数に関係なく were を使うのが原則（口語では主語が I や 3 人称単数なら was を使うこともある）。

**解答**: ①

**日本語訳**: 私があなたの立場なら，SNS の投稿内容にはもっと気をつけるだろう。

---

① could go　② will go　③ am going　④ can go

**解説**
▶ if 節の動詞が過去形なので，主節は「助動詞の過去形 + 原形」になる。

**解答**: ①

**日本語訳**: もしお金がたくさんあったら外国に行けるのだが。

### 問題 160
〈頻出〉

**問題文**

① starts　② started　③ will start　④ are starting

## Theme 47　仮定法過去完了

🔊047

**基本形** If + **S** + had *done* 〜, **S'** + 助動詞の過去形 + have *done* ...

「もし（あの時）〜だったら，…だっただろう（に）」と過去の事実に反することを仮定・想像する構文。入試では仮定法過去完了に関する出題が最も多い。

---

### 問題 161
〈頻出〉

**問題文**
If I (　　) at the station five minutes earlier, I could have caught the train.

① arrived　② had arrived　③ have arrived　④ have been arriving

（南山大）

**解説タイトル**: 仮定法過去完了 —— if 節の形

**解説**
▶ 主節の could have caught に注目して，仮定法過去完了だと判断する。仮定法過去完了の if 節は過去完了形なので，② had arrived が正解。

**解答**: ②

**日本語訳**: 5 分早く駅に着いていたら，その電車に間に合っていただろう。

### 問題 162
〈超頻〉

**問題文**
If I had known you were already here waiting, I (　　) earlier.

（慶應大）

**解説タイトル**: 仮定法過去完了 —— 主節の形

**解説**
▶ if 節の動詞が過去完了形なので，主節は「助動詞の過去形 + have *done*」になる。

**解答**: ②

**日本語訳**: あなたがもうここに来て待っているとわかっていたら，もっと早く来ただろうに。

### 問題 163
〈頻出〉

**問題文**
If I (　　) the seminar last year, I would be able to speak English more fluently now.

① had attended　② have attended　③ should have attended　④ were attending

（立教大）

**解説タイトル**: 仮定法過去・仮定法過去完了のミックス型

**解説**
▶ 「もし（あの時）〜だったら，（今は）…だろうに」という意味を表すには，if 節は仮定法過去完了，主節は仮定法過去という形になる。

ミックス型では問題文中の時を表す副詞要素に注目すること。

本問の場合：if 節〔last year（過去）〕—— 主節〔now（現在）〕

**解答**: ①

**日本語訳**: 去年そのゼミに出席していたら，今ごろはもっと流暢に英語が話せているだろうに。

### Power Up! 29　直説法か仮定法かの識別

**問題**
If it (　　) raining soon, shall we go out for a walk?

① stopped　② stops　③ will stop　④ would stop（センター）

**解説**
「もし雨がやむなら」のように，実際に「ありそうなこと」は，仮定法ではなく「事実をそのまま」述べる直説法で表す。よって，② stops が正解。この stops は，『条件』を表す副詞節中で未来のことを表す現在形。（➡ **Theme 7**）

**答えと和訳**: ② もしもうすぐ雨がやんだら，散歩に行きましょうか。

## Theme 48　should と were to を用いた仮定法

🔊048

**基本形** If + **S** + should *do* 〜, { **S'** + 助動詞の現在形[過去形] + 原形 ... / 命令文 }

**基本形** If + **S** + were to *do* 〜, **S'** + 助動詞の過去形 + 原形 ...

if 節に should や were to を用いて，「万一〔仮に〕〜ならば〔すれば〕」と，未来の実現可能性が低い仮定を表す。

---

### 問題 164

**問題文**
If you (　　) have any questions, please let me know.

① would　② should　③ might　④ could

（県立広島大）

**解説タイトル**: If + S + should *do* 〜「万一〜ならば〔すれば〕」

**解説**
▶ 「（まあ，ないと思うが）万一〜ならば〔すれば〕」のニュアンス。絶対にあり得ないと思われることには用いない。

▶ 本問は主節が命令文なので，if 節は should を用いた仮定法を考える。

**解答**: ②

**日本語訳**: 万一ご質問があれば，お知らせください。

---

### 問題 165

**問題文**
If you (　　) a millionaire, what would you buy?

① are to become　② had become　③ were to become　④ will become

（日本女子大）

**解説タイトル**: If + S + were to *do* 〜「仮に〜ならば〔すれば〕」

**解説**
▶ 「仮に〜ならば〔すれば〕」のニュアンス。絶対にあり得ないと思われることにも，あり得ることにも用いる。

▶ were to の場合は，主節に必ず「助動詞の過去形 + 原形」が来る。

**解答**: ③

**日本語訳**: もしあなたが億万長者になったら，何を買いますか。

## Theme 49　『願望』を表す仮定法

🔊049

---

### 問題 166
〈頻出〉

**問題文**
Dad: The boy has three cats, doesn't he?
Mom: Yeah. That's right. I wonder why he wishes he (　　) another one.

① could have　② have had　③ has　④ does have

（青山学院大）

**解説タイトル**: wish + S + 仮定法過去「S が〜ならよいのに（と思う）」

**解説**
▶ 現在の事実と反対の『願望』なので，wish の後は仮定法過去。wish + S + could 〜 で「S が〜できたらよいのに（と思う）」という意味。

**解答**: ①

**日本語訳**: 父：あの少年は猫を 3 匹飼っているよね？
母：ええ，そうよ。どうして彼は，もう 1 匹飼えたらなんて思っているのかしら。

---

### 問題 167
〈発展〉

**問題文**
I can't hear him. I wish he (　　) a little louder.

① would speak　② will speak　③ speaks　④ can speak

（南山大）

**解説タイトル**: wish + S + would 〜「S が〜してくれたらよいのに（と思う）」

**解説**
▶ 未来のことに対する『願望』を表す。これは，意志未来の will が仮定法過去の形 would になったもの。

**解答**: ①

**日本語訳**: 彼の言うことが聞こえない。もう少し大きな声で話してくれたらよいのにと思う。

### 問題 168

**問題文**
I (　　) we had reserved a table at the restaurant.

① want　② hope　③ wish　④ like

（佛教大）

**解説タイトル**: wish + S + 仮定法過去完了「S が〜ならよかったのに（と思う）」

**解説**
▶ 過去の事実と反対の『願望』なので，wish の後は仮定法過去完了。

〈選択肢〉① want と ④ like は that 節を目的語にとらないので不可。② hope は that 節を目的語にとるが，節内に仮定法は用いないので不可。
> I *hope* I *can* help you.（手伝えればいいのですが）

**解答**: ③

**日本語訳**: そのレストランのテーブルを予約しておけばよかったと思う。

### 問題 169

**問題文**
(　　) I knew her telephone number.

① As to　② If only　③ If ever　④ In that

（摂南大）

**解説タイトル**: If only + S + 仮定法！「S が〜ならよいのに〔よかったのに〕（と思う）」

**解説**
▶ if 節だけで独立的に用いられることが多く，現在〔過去〕の事実と反対の『願望』を表す。If only = I wish と考えればよい。本問は仮定法過去を用いた形。

〈比較〉**If only** I *had listened* to your advice!〔仮定法過去完了〕
（あなたのアドバイスを聞いていればよかったのに！）

**解答**: ②

**日本語訳**: 彼女の電話番号を知っていればなあ。

---

### 問題 170
〈発展〉

**問題文**
I'd rather you (　　) me. I can do it all by myself.

① didn't help　② won't help　③ have not helped　④ were helping

（上智大）

**解説タイトル**: would rather + S + 仮定法「（むしろ）S に〜してほしい」

**解説**
▶ would rather が that 節を伴うとき，節内には仮定法を用いる。

▶ 「wish + 仮定法」とほぼ同意。wish の場合と同様に，that は通例省略される。
> I **would rather you didn't** 〜「〜しないでほしい」の形での出題が多い。

〈補足〉would rather が後ろに原形や than を伴う場合に関しては **82**, **83** を参照。

**解答**: ①

**日本語訳**: 私の手助けをしないでほしい。すべてひとりでできますから。

## Theme 51　as if [as though] + 仮定法

🔊051

---

### 問題 175
〈頻出〉

**問題文**
彼は何でも知っているような口のきき方をする。［2 語不要］
He (as / everything / he / if / knew / know / talked / talks).

（東京理科大）

**解説タイトル**: as if [as though] + 仮定法「まるで〜のように」

**解説**
▶ 「実際にはそうでないが，まるで〜する〔である〕かのように」という意味を表すには，as if の後ろに仮定法過去を用いる。

〈注意〉仮定法は『時制の一致』が行われないので，主節の動詞の時制が現在でも過去でも as if の後ろに来る動詞の形には変わりがない。
> He talks [talked / will talk] *as if* he **knew** everything.

〈比較〉「〜した〔であった〕かのように」の意味のときには仮定法過去完了を用いる。
> He looks **as if** he *had seen* something horrible.
> （彼は何か恐ろしい物を見たような顔つきをしている）

**解答**: He (talks as if he knew everything).［know / talked が不要］

**日本語訳**: He (talks as if he knew everything).［know / talked が不要］

## Theme 52　if を省略した仮定法

🔊052

仮定法における if を省略すると倒置が起こり，疑問文と同じ語順になる。ただし，主語の前に置かれるのは原則として **were / had / should** の 3 つである。

<sup>×</sup>Did I know it ← If I knew it

---

### 問題 177
〈頻出〉

**問題文**
I know you are angry with Chris. (　　) I you, I would break up with him.

① If　② When　③ Were　④ Had

（駒澤大）

**解説タイトル**: Were [<sup>×</sup>Was] I you ← If I were [was] you

**解説**
▶ 本問のように，if を省略した条件節が主節の後に置かれ，しかも前にコンマのない場合は特に注意が必要。この形は英文読解の上でも重要。

**解答**: ③

**日本語訳**: あなたがクリスに腹を立てるのはわかる。もし私があなたなら，彼とは絶交するだろうに。

### 問題 178
〈頻出〉

**問題文**
あなたの住所を知っていたら，手紙を書くこともできたのですが。（成蹊大）
I (address / could / had / have / known / I / written / your / you).

**解説タイトル**: had I known 〜 ← if I had known 〜

**解説**
▶ 本問のように，if を省略した条件節が主節の後に置かれ，しかも前にコンマのない場合は特に注意が必要。この形は英文読解の上でも重要。

**解答**: I (could have written you had I known your address).

**日本語訳**: I (could have written you had I known your address).

### 問題 179

**問題文**

## Theme 53　if 節の代用

🔊053

主語，副詞（句），不定詞などが if 節の代わりをすることがある。特に副詞 **otherwise**「さもなければ」や **without** 〜（➡ **173**）/ **with** 〜（➡ **174**）が重要。if 節がないので，動詞の形（would *do* / would have *done* など）や文脈からその文が仮定法かどうかを判断する。

**解説**

**解答**: (A secret agency would never) let you know their names and real identities.［if が不要］

**日本語訳**: (A secret agency would never) let you know their names and real identities.［if が不要］

### 問題 182
〈頻出〉

**問題文**
I ran to the station. Otherwise, I (　　) the train.

① would have missed　② would been missed　③ would miss　④ missed

（芝浦工大）

**解説タイトル**: otherwise「さもなければ」が if 節の代わりをする場合

**解説**
▶ Otherwise = If I had not run to the station

**解答**: ①

**日本語訳**: 私は駅まで走った。そうでなかったら，電車に乗りそこなっただろう。

### 問題 183

**問題文**
彼がフランス語で話すのを聞くと，フランス人と思うだろう。
(　　) hear him speak in French, you would think him a Frenchman.

（名城大）

**解説タイトル**: to 不定詞が if 節の代わりをする場合

**解説**
▶ To hear him speak in French = If you heard him speak in French

**解答**: To

**日本語訳**: (To) hear him speak in French, you would think him a Frenchman.

### Power Up! 32　if 節の省略
〈発展〉

実際の英文では，if 節あるいは if 節に相当するものが一切なく，主節だけで文を構成している場合がある。その場合にも，助動詞の過去形が仮定法かどうかを判断する目印になる。

The town has changed a lot —— you **wouldn't** recognize it now.
（町はとても変わった。今では（もし来ても）あなたはわからないでしょう）

### Power Up! 33　副詞 otherwise の 3 つの意味

□ 1. 「さもなければ / そうでなかったら」（= if 〜 not）（➡ **182**）
＊ この意味では接続詞と見ることもできる。

□ 2. 「別のやり方で / 違ったふうに」（= differently）
> He thinks **otherwise**.（彼はそうは考えない）

□ 3. 「その他の点では」（= in other ways [respects]）
> The room is small, but **otherwise** it's quite comfortable.
> （部屋は狭いが，その他の点ではとても快適だ）
