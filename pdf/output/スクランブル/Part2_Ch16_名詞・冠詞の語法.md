# 第16章　名詞・冠詞の語法

## Theme 151　不可算名詞［数えられない名詞］の用法

🔊151

日本語の感覚では数えられるように思える名詞でも、英語では数えられないものがある。このような名詞には **a / an** はつかないし複数形にもならない。「多少」は many / few ではなく、much **[a lot of] / little** で表す。ただし、**some / any** は可算名詞にも不可算名詞にも用いられる。

---

### 問題 593
〈頻出〉

**問題文**
Could you give me (　　) about the project?

① an information　② many informations　③ some information　④ a few informations

（中部大）

**解説タイトル**: information「情報」は不可算名詞

**解説**
▶ information は不可算名詞なので、an はつかないし、複数形にもならない。したがって ① an information、② many informations、④ a few informations は不可。正解は、「some + 不可算名詞」「いくらかの～ / 何かの～」の ③ some information。

**解答**: ③

**日本語訳**: その計画について情報をもらえますか。

### 問題 594
〈頻出〉

**問題文**
<u>①Most of the furniture</u> <u>②were</u> <u>③already</u> in the moving van, and the <u>④only thing</u> left was a chandelier.

（早大）

**解説タイトル**: furniture「家具」は不可算名詞

**解説**
▶ most of the A「Aの大部分 / ほとんど」が主語の場合、動詞はAに一致させる。furniture は常に単数形で単数扱いする不可算名詞なので、② were を was に訂正する。

*cf.* **Most of the dogs in the park** *were* playing happily.
（公園にいる犬のほとんどが楽しそうに遊んでいた）

**解答**: ②

**日本語訳**: ［were → was］家具のほとんどはすでに引っ越しトラックに積み込まれ、残っているのはシャンデリアだけだった。

### 問題 595
〈頻出〉

**問題文**
There is not (　　) good news for the economy this year.

① many　② much　③ a few　④ lot of

（獨協大）

**解説タイトル**: news「知らせ」は不可算名詞

**解説**
▶ news は不可算名詞なので、② much が正解。

〈語句〉not much「あまり～でない」は部分否定。（⇨ Power Up! 51）

**解答**: ②

**日本語訳**: 今年、経済について良いニュースはあまりない。

### 問題 596
〈頻出〉

**問題文**
They gave me (　　) on job seeking.

① a piece of advice　② advices　③ an advice　④ many advice

（立命館大）

**解説タイトル**: advice「忠告」は不可算名詞

**解説**
▶ 不可算名詞 advice「忠告」を数える場合は ① a piece of を用いる。

〈選択肢〉③ an「とある、1つの」の後ろは可算名詞の単数形、④ many「多くの」の後ろは可算名詞の複数形が続くので、いずれも不可。② は不可算名詞を複数形にすることはできないので不可。

〈プラスα〉information, furniture, news, baggage などの不可算名詞も a piece of を用いて数を表す。three pieces of furniture「家具3点」

**解答**: ①

**日本語訳**: 彼らは求職活動について私に1つのアドバイスをくれた。

### 問題 597

**問題文**
Playing football is always (　　).

① fun　② a fun　③ funs　④ very fun

（京都産業大）

**解説タイトル**: fun「楽しみ / 面白み」は不可算名詞

**解説**
▶ fun は不可算名詞で、a はつかないし複数形にもならないので、②、③ は不可。

〈選択肢〉強調形は ④ very fun ではなく、a lot of fun / good [great] fun などとする。It would be *a lot of* [*good / great*] *fun*.（それはきっととても楽しい）

〈プラスα〉be fun「楽しい」/ It is fun to *do* [*doing*]「～するのは楽しい」/ have fun「楽しむ」の形で使うことが多い。

**解答**: ①

**日本語訳**: フットボールをするのはいつも楽しい。

---

### 問題 598

**問題文**
The teacher gave us (　　) homework.

① many　② any　③ a few　④ some

（九州産業大）

**解説タイトル**: homework「宿題」は不可算名詞

**解説**
▶ homework「宿題」は work「仕事」と同様に不可算名詞なので、② any か ④ some が考えられるが、肯定文なので、④ some が正解。

**解答**: ④

**日本語訳**: 先生が私たちに宿題を出した。

### 問題 601
〈頻出〉

**問題文**
彼は父親と何年にもわたって仲が悪い。［1語（句）不要］

(related with / on bad / he / his father / has been / terms with) for years.

（中央大）

**解説タイトル**: be on bad terms with ～「～と仲が悪い」

**解説**
〈プラスα〉good や bad 以外にも、friendly「親しい」/ speaking「言葉を交わす」/ visiting「行き来するような」などが用いられる。

She is **on friendly terms with** her neighbors.
（彼女は隣人たちと仲良くしている）

**解答**: He has been on bad terms with his father（related with が不要）

**日本語訳**: （問題文に含む）

### 問題 602
〈頻出〉〈発展〉

**問題文**
Since there was only one washing machine, we had no choice but to (　　) using it.

① change in　② take turns　③ exchange for　④ take place

（獨協医大）

**解説タイトル**: take turns (at [in]) *doing*「交替で～する」

**解説**
▶ この場合の turn は「順番」の意味。交替は1人ではできないので、turns と必ず複数形になる。take turns using it「交替でそれを使う」

〈語句〉have no choice but to *do*「～するしかない」（⇨ Power Up! 20）

**解答**: ②

**日本語訳**: 洗濯機が1台しかなかったので、交替で使わざるを得なかった。

---

### 問題 603
〈頻出〉

**問題文**
スミス教授にはどうにかして連絡を取ります。

I will make contact with Professor Smith by some (　　) or other.

① phone　② means　③ sorts　④ communication

（獨協医大）

**解説タイトル**: means「手段 / 方法」

**解説**
▶ by some means or other「何らかの方法で / 何とかして」

▶ means は「手段 / 方法」の意味では単複同形。単数・複数のどちらにも扱う。

*Every* possible **means** *has* been tried.（単数扱い）
= *All* possible **means** *have* been tried.（複数扱い）
（ありとあらゆる方法が試された）

〈発展〉means には「資産 / 財力」という意味もあるが、その場合は複数扱い。
His **means** *are* small.（彼の財力は乏しい）
live within [beyond] one's **means**「収入に見合った[以上の]生活をする」

**解答**: ②

**日本語訳**: （問題文に含む）

## Theme 156　「予約・約束」を表す名詞

🔊156

---

### 問題 618

**問題文**
［共通の1語を入れよ］

A. Give me your (　　) that you'll never do that again.

（早大）

**解説タイトル**: promise「（一般的な）約束」

**解説**

### Power Up! 136　catch A by the arm の表現

この表現は前置詞によって3つのタイプに分けられる。

(1) <sup>*</sup>catch [take / hold / grasp] **A by the arm**「Aの腕をつかむ」（⇨ 621）
　　pull **A by the sleeve**「Aの袖を引く」

(2) <sup>**</sup>strike [hit] **A on the head**「Aの頭をなぐる」
　　<sup>**</sup>pat [tap] **A on the shoulder**「Aの肩を（軽く）たたく」（⇨ 622）
　　kiss **A on the cheek**「Aの頬にキスをする」

(3) look [stare] **A in the eye(s)** [face]「Aの目［顔］をじっと見る」（⇨ 623）

- <sup>*</sup>catch：動いているものをつかむ / take：動かないものを手に取る / hold：つかんでいる［持っている］ / grasp [grip]：しっかりとつかむ
- <sup>**</sup>strike：手や棒で強く打つ / hit：ねらって強く打つ / pat [tap]：手のひらで優しく［軽く］たたく

## Theme 158　注意すべき冠詞の位置

🔊158

---

### 問題 624
〈頻出〉

**問題文**
I've never seen (　　) movie.

① a such funny　② so funny a　③ a so funny　④ so a funny

（獨協医大）

**解説タイトル**: so + 形容詞 + a [an] + 名詞

**解説**
▶ 副詞 so の場合は、形容詞が冠詞 a に先行して「so + 形容詞 + a [**an**] + 名詞」の語順になるので、正解は ② so funny a。

〈プラスα〉so を用いるのは硬い表現なので、日常的な書き言葉や会話では such を用いて、such a funny movie とするほうがふつう。

〈発展〉複数名詞の場合は <sup>×</sup>so funny movies とは言わない。
°such funny movies はよい。

**解答**: ②

**日本語訳**: 私はこんなに面白い映画を見たことがない。

### 問題 626
〈頻出〉

**問題文**
I have never seen (　　) starry sky.

① a so beautiful　② such a beautiful　③ so a beautiful　④ a such beautiful

（高知大）

**解説タイトル**: such + a [an]（+ 形容詞）+ 名詞

**解説**
▶ 形容詞としての such「そのような～ / そのように…な～」は、冠詞 a [an] の前に置かれ、「**such** + a [**an**]（+ 形容詞）+ 名詞」の語順になる。

**解答**: ②

**日本語訳**: 私はこんなに美しい星空を見たことがない。

### Power Up! 137　注意すべき冠詞の位置

(1) ふつう冠詞は名詞の前に置かれ、名詞に形容詞がついた場合には、
「**a [an]** + 形容詞 + 名詞」の語順になる。a pretty girl「かわいい女の子」

(2) 形容詞に副詞 very がついても冠詞は一番前に置かれ、
「**a [an]** + very + 形容詞 + 名詞」の語順になる。a very pretty girl

(3) 副詞が so / as / too / how の場合は「**so [as / too / how]** + 形容詞 + a [an] + 名詞」の語順になる。文語表現だが入試には頻出。so [as / too / how] pretty a girl

(4) such が後ろに名詞を伴うと、「**such** + a [an]（+ 形容詞）+ 名詞」の語順になる。
such a pretty girl（= so pretty a girl）

## Theme 159　a friend of mine などの形

🔊159

所有格は a / this / that / some / any / most / no などと並べて使えないため、
