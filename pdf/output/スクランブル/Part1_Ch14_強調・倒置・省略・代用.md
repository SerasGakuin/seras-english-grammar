# 第14章　強調・倒置・省略・代用

## Theme 118　強調構文

🔊118

強調したい文の要素を **It is** と **that** ではさんで強調する表現。
(1) 強調できる要素は、主語、目的語などとして働く名詞（句・節）か副詞（句・節）だけである。動詞と形容詞は強調できない。
(2) 強調されるものが「人」なら **who [whom]**、「人以外」なら **which** を、それぞれ that の代わりに用いることもある。

---

### 問題 452
〈頻出〉

**問題文**
君に解いてもらいたいのはほかでもないこの問題なんだ。
It is this question (solve / I / to / you / want / that).

（千葉工大）

**解説タイトル**: It is ... that 〜「〜なのは…である」

**解説**
▶ 元の文は I want you to solve this question. である。本問は目的語の this question を It is と that ではさんで強調したもの。

**解答**: It is this question that I want you to solve.

**日本語訳**: 君に解いてもらいたいのはほかでもないこの問題なんだ。

---

### 問題 453
〈頻出〉

**問題文**
電話ができなかったのは、充電が切れていたからです。
It is (because / battery / that / my phone / ran / out of) I couldn't make a call.

（千葉工大）

**解説タイトル**: It is [was] because ... that 〜「〜なのは…だ［だった］からだ」

**解説**
▶ 本問は、I couldn't make a call because my phone ran out of battery. の副詞節 because my phone ran out of battery を強調した文。

**解答**: It is because my phone ran out of battery that I couldn't make a call.

**日本語訳**: 電話ができなかったのは、充電が切れていたからです。

---

### 問題 454
〈頻出〉

**問題文**
My father scolded me, but it was my sister (　　) was to blame.

① for　② to　③ which　④ who

（千葉工大）

**解説タイトル**: It is ... who 〜「〜なのは…である」

**解説**
▶ it was の後が「人」なので、(　　) に ④ who を入れれば、my sister を強調することになる。

**解答**: ④

**日本語訳**: 父は私をしかったが、とがめられるべきなのは姉［妹］だった。

---

### 問題 455

**問題文**
あなたが会いたいのは誰ですか。
(see / want / you / it / that / to / is / who)?

（尾道市大）

**解説タイトル**: 疑問詞 + is it that 〜?

**解説**
▶ 強調されるものが疑問詞のこともある。その場合、疑問詞が文頭に出て it と is が入れ替わり、「疑問詞 + is it that 〜?」の形になる。

**解答**: Who is it that you want to see?

**日本語訳**: あなたが会いたいのは誰ですか。

---

> **Power Up! 87　強調構文と形式主語構文の見分け方**
>
> It is ... that 〜 の「...の部分の品詞」に着目する。
> (1) It is + 副詞 + that 〜 → 強調構文
> 　It was **yesterday** that he met the girl in the park.
> (2) It is + 形容詞 + that 〜 → 形式主語構文
> 　It is **certain** that he met the girl in the park yesterday.
> (3) It is + 名詞 + that + 名詞が欠けた文 → 強調構文
> 　It was **he** that met the girl in the park yesterday.
> 　（that の後ろに主語が欠けている）
> 　It was **the girl** that he met in the park yesterday.
> 　（that の後ろに目的語が欠けている）
> (4) It is + 名詞 + that + 完全な文 → 形式主語構文
> 　It is **a fact** that he met the girl in the park yesterday.
> 　（that の後ろは完全な文）

## Theme 119　語句による強調

🔊119

---

### 問題 456

**問題文**
A: You are the (　　) person I wanted to see.
B: Oh, how can I help you?

① so　② too　③ very　④ much

（追手門学院大）

**解説タイトル**: very + 名詞「まさにその… / ちょうどその…」―― 名詞の強調

**解説**
▶ very が形容詞として名詞を強調する用法。
　at that **very** moment「ちょうどその時」, in this **very** town「まさにこの町で」
〈選択肢〉① so, ② too は副詞なので、直接名詞を修飾できない。④ much には形容詞の用法もあるが、不可算名詞を修飾するので、本問では不適切。

**解答**: ③

**日本語訳**: A：あなたは私がまさに会いたいと思っていた人です。
B：そうですか、何かできることはありますか。

---

### 問題 457

**問題文**
私は、本当にうさぎが私の机に飛び乗るのを見たのです。
I (　　) see a rabbit jumping on my desk!

① did　② must　③ would　④ had to

（名古屋学院大）

**解説タイトル**: do [does / did] + 動詞の原形「本当に〜する［した］」―― 動詞の強調

**解説**
▶ 動詞を強調する（文の内容が事実であることを強調する）助動詞 do。

**解答**: ①

**日本語訳**: 私は、本当にうさぎが私の机に飛び乗るのを見たのです。

---

### 問題 458

**問題文**
How (　　) did you know that?

① in earth　② in the earth　③ in world　④ on earth

（清泉女子大）

**解説タイトル**: on earth「いったい（全体）」（= in the world）―― 疑問詞の強調

**解説**
▶ 疑問詞の直後に on earth / in the world / ever などの語句をつけて、疑問詞を強調することがある。
　本問 = How *in the world* did you know that? = How *ever* did you know that?

**解答**: ④

**日本語訳**: いったいどうしてあなたはそれを知ったのですか。

## Theme 120　強調のための倒置

🔊120

強調のために **never / little / rarely / seldom** などの否定語を文頭に出すと、「主語＋動詞」の語順が「（助）動詞＋主語」と逆転する（疑問文と同じ語順になる）ことを倒置という。

Never *have I seen* such a sight. （← I have **never** seen such a sight.）
（そんな光景は見たことがない）

---

### 問題 459
〈頻出〉

**問題文**
こんなすばらしい賞品をもらえるとは夢にも思いませんでした。
Little (did / dream / of / getting / I) such a wonderful prize.

（佛教大）

**解説タイトル**: 否定の副詞（句）を文頭に出す場合

**解説**
▶ little はふつう「ほとんど〜ない」という弱い否定を表すが、これを文頭に出した倒置文では「少しも〜ない」という強い否定を表す。

**解答**: Little did I dream of getting such a wonderful prize.

**日本語訳**: こんなすばらしい賞品をもらえるとは夢にも思いませんでした。

---

### 問題 460
〈頻出〉

**問題文**
昨日になってやっと、私は彼女が本当に言いたかったことがわかりました。
(yesterday / only / finally / what / actually / did / she / understand / I) wanted to say.

（白百合女子大）

**解説タイトル**: only を含む副詞句・副詞節を文頭に出す場合

**解説**
▶ 副詞 only は否定に近い意味を持つので、only を含む副詞句・副詞節を文頭に出す場合も、その後では倒置が起こり、疑問文と同じ語順になる。
〈重要〉「only + 時の副詞（句・節）」が強調のために文頭に置かれる場合が重要である。「〜の時になって初めて［やっと］」と訳す。
　**Only when it's dark** *can you* see the stars.
　（暗くなって初めて、星が見える）

**解答**: Only yesterday did I finally understand what she actually wanted to say.

**日本語訳**: 昨日になってやっと、私は彼女が本当に言いたかったことがわかりました。

---

### 問題 461

**問題文**
Not only (　　) his smartphone yesterday, but he also left his watch at home.

① did he forget　② he forgot
③ he has forgotten　④ is he forgetting

（神奈川大）

**解説タイトル**: Not only ... but (also) 〜 が文と文を結ぶ場合

**解説**
▶ not only ... but also 〜 が文と文を結ぶ場合に、not only が文頭に出ると、not only の後は倒置が起こり、疑問文と同じ語順になる。
〈選択肢〉④は倒置形だが現在進行形であり、過去のことを表せないので不可。

**解答**: ①

**日本語訳**: 昨日彼はスマートフォンを忘れただけでなく、時計も家に忘れてきた。

## Theme 121　慣用的な倒置

🔊121

---

### 問題 462
〈頻出〉

**問題文**
Susan likes chocolate and so (　　) Hiroshi.

① do　② does　③ is　④ likes

（青山学院大）

**解説タイトル**: so + 助動詞［be 動詞］+ S「S もまたそうだ」

**解説**
▶ 前の肯定の内容を受けて、「S もまたそうだ」という場合、so + 助動詞［be 動詞］+ S の倒置形を用いる（助動詞は do や完了形の have も含む）。
▶ 本問では likes chocolate を受けて、so does Hiroshi となる。一般動詞を受ける場合は do [does / did] を使うが、be 動詞を受ける場合は is [am / are / was / were] を使う。
　"I'm a student." "**So am I.**"（「私は学生です」「私もです」）

**解答**: ②

**日本語訳**: スーザンはチョコレートが好きで、ヒロシもまたそうだ。

---

### 問題 463
〈頻出〉

**問題文**
We don't want to go there, and (　　) they.

① either do　② so do
③ neither do　④ neither don't

（関西学院大）

**解説タイトル**: neither + 助動詞［be 動詞］+ S「S もまたそうでない」

**解説**
▶ 前の否定の内容を受けて、「S もまたそうでない」という場合、neither + 助動詞［be 動詞］+ S の倒置形を用いる。462 の否定形。

**解答**: ③

**日本語訳**: 私たちはそこに行きたくないし、彼らも行きたくない。

---

### 問題 464
〈頻出〉

**問題文**
We don't want to give it up, (　　) do they.

① as　② either　③ nor　④ so

（武庫川女子大）

**解説タイトル**: nor + 助動詞［be 動詞］+ S「S もまたそうでない」

**解説**
▶ neither と同様、前の否定の内容を受けて、「S もまたそうでない」の意味を表す。neither と nor の違いは、**Power Up! 88** を参照。
〈重要〉nor の後ろに節を続ける場合は、「nor + 疑問文の語順」に並べる。
　They didn't understand the instructions, **nor** *did they ask for help*.
　（彼らは指示を理解していなかったし、助けを求めることもしなかった）

**解答**: ③

**日本語訳**: 私たちはあきらめたくないし、彼らもそうだ。

---

> **Power Up! 88　neither と nor の使い分け**
>
> **問題**　最初のはよくないし、2番目もよくない。
> The first isn't good and (　　) is the second.（京都産業大）
> ① either　② neither　③ nor　④ or
>
> **解説・答え**　nor は接続詞なので、この問題のように接続詞 and がある場合、③ nor は使えない。正解は副詞の ② neither。

## Theme 122　that 節の代用の so と not

🔊122

hope / be afraid / think / suppose などの目的語が、肯定の that 節なら **so** を、否定の that 節なら **not** を代用する。

---

### 問題 465

**問題文**
A: Will you be able to come to the concert?
B: I hope (　　).

① so　② too　③ that　④ I do

（センター）

**解説タイトル**: I hope so.「そうであることを願う / そうだといいね」

**解説**
▶ 本問では so は hope の目的語である肯定の that 節の代用。
　I hope so. = I hope that I will be able to come to the concert.

**解答**: ①

**日本語訳**: A：コンサートに来られる？　B：行けたらいいな。

---

### 問題 466
〈頻出〉

**問題文**
A: Do you think he's coming?
B: I am afraid (　　).

① either　② it　③ neither　④ not

（センター）

**解説タイトル**: I'm afraid not.「残念ながらそうでないと思う」

**解説**
▶ 本問は am afraid の目的語である否定の that 節を代用する ④ not が正解。
　I am afraid not. = I am afraid that he is not coming.
〈発展〉think / suppose など は I think [suppose] not. = I don't think [suppose] so.「そうではないと思う」の2種類の形が使えるが、think は後者のほうがふつう。

**解答**: ④

**日本語訳**: A：彼は来ると思いますか。　B：残念ながら来ないと思います。

---

### 問題 467
〈頻出〉

**問題文**
A: Frank drives much too fast. Someday he'll have a terrible accident.
B: Oh, (　　).

① I don't hope so　② I hope not
③ I'm not afraid so　④ I'm afraid not

（センター）

**解説タイトル**: I hope not.「そうでないことを願う」

**解説**
▶ hope / be afraid はそれぞれ、I hope not. / I'm afraid not. の形を使う。
　① I don't hope so. や ③ I'm not afraid so. の形は不可。正解は、② I hope not（= I hope that he will not have a terrible accident）。
〈選択肢〉④ I'm afraid not. だと「残念ながら彼はひどい事故を起こさないと思う」の意味になるので不適切。

**解答**: ②

**日本語訳**: A：フランクはあまりにもスピードを出しすぎる。いつかひどい事故を起こすでしょうね。
B：そうならないことを願います。

---

> **Power Up! 89　I hope 〜 と I am afraid 〜 の4つの用法**
>
> Will it rain?（雨が降るでしょうか）
> ① ― I hope so.（降ってほしいですね）
> ② ― I hope not.（降らなければいいなあ）
> ③ ― I am afraid so.（残念ながら降るでしょうね）
> ④ ― I am afraid not.（残念ながら降らないでしょうね）
> ＊ so = that it will rain / not = that it will not rain

## Theme 123　副詞節中の［S + be 動詞］の省略

🔊123

副詞節において「S + be 動詞」が省略されることがある。特に副詞節中の主語が主節の主語と同一である場合に多い。

---

### 問題 468
〈頻出〉

**問題文**
A plane crashed while (　　) to land in Portland.

① attempt　② attempts
③ attempted　④ attempting

（神田外語大）

**解説タイトル**: while *doing*「〜している間」

**解説**
▶ 本問は while it was attempting to land in Portland の it was が省略された形。

**解答**: ④

**日本語訳**: ポートランドに着陸しようとしている時に、飛行機は墜落した。

---

### 問題 469
〈頻出〉

**問題文**
She can explain the rules of chess if (　　).

① asking　② asked　③ she asked　④ having asked

（関西大）

**解説タイトル**: if *done*「もし〜されたら」

**解説**
▶ 本問は if she is asked の she is が省略された形。

**解答**: ②

**日本語訳**: 求められれば、彼女はチェスのルールを説明できる。

---

> **Power Up! 90　副詞節中の［S + be 動詞］の省略例**
>
> ① Work hard **while** (you are) young.（若いうちに一生懸命働きなさい）
> ② They were good friends **when** (they were) in school.
> （彼らは学生時代仲良しだった）
> ③ **Though** (I am) tired, I'll come with you, **if** (it is) necessary.（➡ **Power Up! 91**）
> （疲れているけれど、必要なら一緒に行きますよ）
> ④ **Once** (it is) signed, the contract will be valid until the end of March.
> （いったん署名したら、その契約は3月末まで有効になるだろう）（➡ 421）

## Theme 124　慣用的な省略

🔊124

if 節では、主節の主語と if 節の主語が違う場合でも、慣用的に「S + V」が省略されることが多い。if any / if not など挿入句として使われる重要表現が多いので要注意。

---

### 問題 470

**問題文**
There are few, (　　), such good men.

① all but　② at all　③ if any　④ if possible

（東北学院大）

**解説タイトル**: if any「たとえあるにしても」

**解説**
▶ 通例 few [little] などの次に来て「たとえあるにしても」という「譲歩」の意味を表す。
〈選択肢〉① all but「〜の他はみな / ほとんど」, ④ if possible「できたら」
〈発展〉if any には「もしあるならば」の意味もある。
　Correct errors, **if any**.（もし誤りがあるならば、訂正しなさい）
　省略を補えば、Correct errors, if (there are) any (errors). となる。

**解答**: ③

**日本語訳**: そんないい人たちは、たとえいるにしても、ごく少ない。

---

### 問題 471

**問題文**
Contrary to popular belief, aerobic exercise does not make you tired and sleepy. If (　　), it helps you concentrate more.

① anything　② everything　③ nothing　④ something

（立教大）

**解説タイトル**: if anything「どちらかと言えば / むしろ」（= rather）

**解説**
〈発展〉if any と同じ使い方もある。
　Little, **if anything**, can be done to improve the situation.
　（状況を改善するためにできることは、たとえあったとしても、ほとんどない）

**解答**: ①

**日本語訳**: 一般的な考えに反して、有酸素運動によって疲れて眠くなることはない。どちらかというと、より集中するのに役立つ。

---

### 問題 472

**問題文**
He seldom, (　　), goes to the movies.

① if any　② if ever　③ if impossible　④ if not

（京都産業大）

**解説タイトル**: if ever「たとえあるにしても」

**解説**
▶ 通例 seldom / rarely「めったに〜ない」の次に来て「たとえあるにしても」の意味になる。

**解答**: ②

**日本語訳**: 彼は、たとえあるにしても、めったに映画を見に行かない。

---

### 問題 473

**問題文**
My brother loves baseball. He's an enthusiastic, (　　) not a gifted, player.

① as　② if　③ or　④ so

（センター）

**解説タイトル**: A, if not B「B とは言わないまでも A / B でないにしても A」

**解説**
▶ A, B には通例、名詞、形容詞、副詞が来る。
▶ 本問の場合、挿入されている (　　) not a gifted の (　　) に ② if を入れれば、if not a gifted (player) = if he is not a gifted (player)「天賦の才能がある（選手）とは言わないまでも」の意味になり、文意が通る。
▶ 本問は、主節の主語 He（= My brother）と if 節の主語は一致している。
〈発展〉if not には「もしそうでなければ」の意味もある。
　We should leave now; **if not**, we'll miss the train.
　[if not = if we don't leave now]
　（今出発すべきです。そうでなければ、電車に乗り遅れます）

**解答**: ②

**日本語訳**: 私の兄［弟］は野球が大好きです。彼は天賦の才能があるとは言わないまでも、熱心な選手です。

---

> **Power Up! 91　慣用的な省略**
>
> □ **if any**　「たとえあるにしても」（➡ 470）
> □ **if anything**　「どちらかと言えば / むしろ」（➡ 471）
> □ **if ever**　「たとえあるにしても」（➡ 472）
> □ **A, if not B**　「B とは言わないまでも A / B でないにしても A」（➡ 473）
> □ **if possible**　「できるなら / できたら」
> 　Please arrive early, **if** (it is) **possible**.
> 　（できれば早めに到着してください）
> □ **if necessary**　「必要ならば」
> 　You can use my car, **if** (it is) **necessary**.
> 　（必要なら私の車を使ってください）
> □ **if so**　「もしそうならば」
> 　Did I say something wrong? **If** (I did) **so**, I apologize.
> 　（何か悪いことを言いましたか。もしそうならば謝ります）
