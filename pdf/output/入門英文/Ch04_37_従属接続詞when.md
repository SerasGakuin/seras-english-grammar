## 精講

> (In March), {S}❶the AI-based computer program AlphaGo, (❷developed by DeepMind), {V}shocked {O}the world [❸{従接}when {S}it {V}defeated {O}South Korean *go* grandmaster Lee Sedol (in a five-game match of the ancient board game [{S}❹{関代}that {V}requires {O}deep insight])].

In March, the AI-based computer program AlphaGo, developed by DeepMind, shocked the world when it defeated South Korean *go* grandmaster Lee Sedol in a five-game match of the ancient board game that requires deep insight.
### ❶　the AI-based computer program AlphaGo は？　〔名〕〔詞〕

　the AI-based computer program AlphaGo は，the AI-based computer program「AIを基にしたコンピュータプログラム」と，その呼び名である AlphaGo を並べて書いていて，全体で1つの名詞であり，これが文の主語になります。

### ❷　developed と shocked はどちらが動詞？　〔分詞〕〔動詞〕

　develop には自動詞「発達する」と他動詞「〜を開発する」の用法がありますが，本文では後ろに by DeepMind「ディープマインド社によって」があるので，受動の意味の過去分詞の形と推測できますね。さらに，主語と動詞の間にコンマが打たれることは通常はないと考えられます。以上より，developed by DeepMind は，その前の主語（the AI-based computer program AlphaGo）に対する説明を追加した部分です。このように，**主語の直後にコンマで挟まれた現在分詞・過去分詞は，文法的には分詞構文の扱い**となります。

　shocked はどうでしょうか？　日本語では「あんなことを言われて，私，ショックだった！」のように使いますが，英語の shock は他動詞で「〜に衝撃［ショック］を与える」という意味です。shocked the world は「世界に衝撃を与えた」となります。shocked が本文の中心の動詞です。

部分訳　ディープマインド社によって開発されたアルファ碁は世界に衝撃を与えた

### ❸　when の役割は？　**Point**　〔接続詞〕

　ここまでの文では「アルファ碁が世界に衝撃を与えた」でしたね。ということは，1つの完全な文になっているわけですから，when ... は修飾語だとわかります。つまり **when は「〜するときに」という意味の接続詞で，後ろの文を副詞にまとめる動き**です。このように主節にある節と結びつける役割をす

る接続詞を従属［従位］接続詞といいます。**従属接続詞には(1)名詞節を作るものと (2)副詞節を作るもの**があります。副詞節を作る接続詞は数多くありますが，主なものを取り上げてみます。

| | |
|---|---|
| 時 | □ when / as「〜するときに」　□ while「〜する間に」 |
| | □ until[till]「〜するまで」　□ since「〜して以降」 |
| 理由 | □ because / since / as「なぜなら〜，〜なので」 |
| 条件 | □ if「もし〜なら」　□ unless「〜の場合を除いて」 |
| 譲歩 | □ though[although]「〜だけれども」 |

　① as は「だいたい等しい」が基本の意味です。文脈によって「〜のように」（様態），「〜しながら」（時），「〜なので」（理由）の意味を持ちます。
　① while は文脈によって譲歩や対比の意味にもなります。

　主語の it は the AI-based computer program AlphaGo を指し，他動詞 defeat「〜を打ち破る」の目的語は，South ... Sedol です。ここでも複数の名詞が並列されていますが，South Korean / go grandmaster / Lee Sedol のように，前から素直に意味をとっていきます。

部分訳　それが韓国の碁の名人イ・セドルを打ち破ったときに

### ❹　that の働きは？　〔関係詞〕

　that の後ろには ᵥrequires ₒdeep insight がありますが，主語に当たる語がありません。よって that は主格の関係代名詞だとわかります。先行詞 the ancient board game で，これは前出の go「碁」を指しています。

　なお，**〈the＋名詞＋関係代名詞節〉で「〜の唯一の…」**の意味になることがあります。本文でも，もしかすると筆者は「深い洞察を必要とする古くからのボードゲームはただ1つしかない」と思っているのかもしれませんね。

## 一言

　竹岡が工学部の学生の頃，「いったん取った駒をもう一度使うという点で，無限に近い可能性があるから，将棋でコンピュータが勝てるわけない」というのが，同級生達の一致した見解でした。しかし，将棋でもその予想は見事に裏切られました。今後，様々な分野でAIが人間を超越するかもしれません。AIに負けない何かを身につけていくことが大切なのだと思います。

## 解答例

3月，ディープマインド社によって開発された人工知能を基にしたコンピュータプログラムであるアルファ碁は，深い洞察力を必要とするその古くからのボードゲームの五番勝負で韓国の碁の名人であるイ・セドルを打ち破り世界に衝撃を与えた。
