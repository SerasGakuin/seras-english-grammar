## 精講

(According to Dr. Carter), ₛᵥ❶**it's** ᴄ**not easy** — even for scientists — (to ₍ᵥ₎define ₍ₒ₎sleep precisely). ❷Basically, [❸従接when ₛᵥwe're ᴄasleep] ₛᵥ**we're** ❹hardly ᴄ**aware** (of the things around us) 等接and ₛ**we** ᵥ**don't move** much.

### ❶ この it は何？　〔不定詞〕

it's not easy — even for scientists — を読むと「それは簡単ではない。科学者にとってさえ」とあります。ここまで読むと当然「何が簡単ではない？」という気持ちになりますね。このitは形式主語で，このあとに**「真主語」と呼ばれる不定詞 to (V)があるだろう**と予測してください。これは，頭でっかちの文を避け，なおかつ「一番言いたいことを文末に置く」という，"end focus"「文末焦点」の法則によるものです。to define sleep preciselyとは「睡眠を厳密に定義すること」の意味です。ダッシュに挟まれたeven for scientistsが不定詞の意味上の主語で，「～にとって」ではありません。

部分訳　カーター博士によれば，睡眠を厳密に定義することは，科学者にとってさえ簡単ではない。

### ❷ Basically はどこにかかる？　〔副詞〕

副詞の中には「語修飾」の副詞と，「文修飾」の副詞があります。両者は辞書では明確に区別されているので確認してみてください。わかりやすい例はprobably「おそらく」で，これは何かの単語を修飾しているのではなく，**文全体を修飾**しています。こういう副詞を「文修飾の副詞」というのです。**文修飾の副詞は文頭に置く**のが基本で，しばしば後ろにコンマを伴います。

本文のbasicallyは「語修飾」語としては「基本的に」という意味ですが，同時に文修飾語として「**基本的に(言えば)，実のところ**」という意味も持ちます。本文では，basicallyの直後に，basicallyが修飾するような形容詞や副詞が見当たりません。よってbasicallyは文修飾語だとわかります。

第2文全体は，〈when S＋V', S＋V₁ 等接and S＋V₂〉(Sは共通でwe)です。内容から考えて，〈when S＋V'〉は〈S＋V₁〉 we're hardly aware of the things around usだけでなく〈S＋V₂〉 we don't move muchも修飾しています。つまり「眠っているときには」，「周りのことを意識することがほとんどなく」かつ「あまり動かない」ということです。

### ❹ hardly の意味は？　**Point**　〔否定〕

以下の表のように，形容詞と副詞で形が同じ語があります。

| late |  | 遅い |  |  | 遅く |
|---|---|---|---|---|---|
| fast | 形 | 速い |  | 副 | 速く |
| hard | 容 | かたい，困難な |  | 詞 | 熱心に |
| early | 詞 | 早い |  |  | 早く |
| high |  | 高い |  |  | 高く |

注意すべきはhard, late, highに-lyがつくと，**違う意味になる**ということです。hardlyは「ほとんど～ない」，latelyは「近頃」，highlyは「おおいに」という意味に変化します。本文の**hardlyも「ほとんど～ない」という否定語なのでnotと同じ位置**にありますね。

the things around usのaroundは前置詞で，「～の周りに」の意味です。よって「私達の周りのもの」の意味となります。theがついているのは後ろの修飾語(around us)によって意味が限定され特定可能だ(と筆者が考えている)からです。the name of my catのtheと同じです。

## 一言

あなたは「一番言いたいこと」を「先に言う」派ですか，それとも「あとで言う」派ですか？　英語はおそらく両者の中間に属する言語だと思います。否定語はできるだけ文の先頭近くに置くことが好まれます。自分の立場がyesなのかnoなのかを明確にしたいということですね。ところが，それ以外では，一番言いたいことは文末に置く傾向にあります。これを"end focus"「文末焦点」という言い方をする場合があります。「形式上の主語」と「真の主語」という関係もこれに似た，「本当に言いたいことをあとに回す」という心慣に依拠したものなのでしょうね。

## 解答例

カーター博士によれば，睡眠を厳密に定義することは，科学者にとってさえ簡単ではない。基本的に，眠っているときには，周りのものをほとんど意識しないしあまり動かない。
