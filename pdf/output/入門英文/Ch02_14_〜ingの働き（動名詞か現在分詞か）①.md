## 精講

> {S}The earliest evidence (for {s}modern humans {v}❶using {o}❷rainforest environments) {V}❸was (from around 45,000 years ago) (in Niah Cave, Borneo). {S}These regions {V}❹seem {C}an unlikely place (❺for {s}early humans to {v}live).

The earliest evidence for modern humans using rainforest environments was from around 45,000 years ago in Niah Cave, Borneo. These regions seem an unlikely place for early humans to live.

### ❶　using はどこにつながる？　**Point**　〔動名詞〕

　ここは以下の2つの可能性が考えられます。

　(1)　using を現在分詞として，名詞 modern humans を後ろから修飾していると考えるものです。その前の The earliest evidence とつなげると，「熱帯雨林という環境を用いる現生人類の最も古い証拠」つまり「現生人類の証拠」となりますが，これではよく意味がわかりません。

　(2)　using を動名詞として，using rainforest environments という**動名詞の句**に，**modern humans という意味上の主語がついた形**と考えるものです。この場合には「現生人類が熱帯雨林という環境を用いること」となり，前とつなげると「現生人類が熱帯雨林という環境を用いた証拠」の意味になるので(2)を選択します（例　基本的に準動詞の動名詞や分詞は（動詞ではないので）**時制を持たず，「いつの時点のことか」は文脈に応じて判断**します。本文では「証拠」と言っている以上過去の話ですから「用いた」が適訳です）。**〈名詞＋(V) ing〉は現在分詞と動名詞の2つの可能性がある**ので，毎回立ち止まって意味をよく考えてくださいね。基本の～ ing の働きは次のとおりです。

| | 文中での働き | 文中での要素 |
|---|---|---|
| 動名詞 ～ ing | 名詞になる | 主語，補語，動詞や前置詞の目的語になる |
| | 形容詞になる | 名詞を前や後ろから修飾する |
| 分詞 ～ ing | 副詞になる | 文全体や名詞以外を修飾する |
| | | 分詞構文が多い。SVOC の C になることも |

　early は「ある時間軸の前半」という意味です。early in the morning なら「午前中のなかの前半」→「朝早く」です。本文では，「最も早くに見つかった」→「最古の」という意味です。

### ❷　〈名詞＋名詞〉って何？　〔名詞〕

　英語では〈名詞＋名詞〉で，**前の名詞が後ろの名詞を修飾する**ことがあります。例えば an enemy ship は「敵」＋「船」ですが，「敵の船」のことです。本文では「熱帯雨林という環境」です。なお，environments と複数形になっているので，そのような環境がいくつか存在したことがわかります。

### ❸　〈be 動詞＋from ～〉って何？　〔前置詞〕

　〈be 動詞＋from ～〉は「～から来た」の意味です。「私は京都市から来ました（出身です）」は，I'm from Kyoto. と言います。本文では，「およそ 45,000年前から来た」とあるので，「ボルネオのニア洞窟の中にあるおよそ 45,000年前のものだった」とすると日本語らしいですね。

### ❹　seem の意味は？　〔動詞〕

　seem は「～と思われる」という意味です。日本語では受動態の感じですが，英語では受動態にはしません。These regions seem ～「こうした地域は～と思われる」の「～」に当たるのが an unlikely ... live の部分です。なお，こうした地域とは「ニア洞窟が存在するような熱帯雨林という環境」のことですね。

### ❺　for は「～にとって」？　〔不定詞〕

　an unlikely place「ありそうもない場所」とあるので，「一体何がありそうもないのだろう？」と思って読み進めます。for early humans だけなら「初期の人間にとって」となりますが，直後に to live という to 不定詞があるので，読み方を変更します。つまり**〈for *A* to (V)〉で「A が～する」**という意味です。***for A* は「to 不定詞の意味上の主語」**と呼ばれるものです。さらに，for early humans to live は前の **an unlikely place を修飾する形容詞用法の to 不定詞**です。ここまで読んでようやく，「ありそうにない」ことの具体的内容がわかりました。つまり「初期の人間が住みそうにもない」ということですね。

## 一言

　本文によると，熱帯雨林で生き抜くには技術の改良・革新と高度な狩猟技術を要するので，この発見は考古学者を驚かせたとあります。彼らは熱帯雨林という環境をどのように利用し生きていたのでしょうね？

## 解答例

現生人類が熱帯雨林という環境を利用していたことの最古の証拠は，ボルネオのニア洞窟の中にあったおよそ 45,000年前のものである。こうした地域は初期の人間が住んでいそうにはない場所のように思われる。
