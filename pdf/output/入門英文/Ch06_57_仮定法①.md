# 57　仮定法①

→ 問題　別冊 64ページ

## 精講

> [❶{従接}If {S}every ❷choice [{S}you {V}made] {V}carried {O}consequences [{S}you {V}would bear ❸not once {等接}but countless times]], {S}you {V}would certainly think hard (before {v}making {o}a ❺decision [{S}you {V}might later regret]).

If every choice you made carried consequences you would bear not once but countless times, you would certainly think hard before making a decision you might later regret.

### ❶ この If は？　**Point**　〔仮定法〕

この文は仮定法過去で書かれています。仮定法過去の基本的な形は〈**If S'＋動詞の過去形 ..., S would[could / might / should]＋動詞の原形 ...**〉です。形は過去ですが，**過去のことを示しているわけではありません**。「もし雨が降ったら家にいよう」と「もし宝くじで1億当たったらポルシェを買おう」では，現実になる可能性はずいぶんと異なりますね(前者は日常的なことだが，後者は人生ではほぼ起こらない)。日本語ではどちらの場合も「～たら」で，形の面では区別しません(古文の世界では，後者は反実仮想という形で区別しました)。英語でも，**その可能性を本人がどう見積もっているか**で，仮定法で表すかどうかが決まります。「可能性が低いな」と思えば仮定法を使います。

### ❷ choice you はどうなっている？　〔関係詞〕

2行目のコンマで区切られた後のyou would ... が，本文の主節です。よって，If節はtimesまでだと判断できます。

every choice you madeは，choice **you**と(代)名詞が連続していること，また**madeの目的語にあたる名詞がない**ことから，**目的格の関係代名詞の省略**を考えます。つまり，every choice which[that] you madeからwhich[that]が省略された形です。このevery choice you madeがif節の主語です。後ろのcarried consequences「結果を運んだ」がif節の〈V＋O〉になっています。この後ろにあるyou would bearは，consequences **you**と名詞の後ろに代名詞が連続していて，bearが他動詞であることを考えれば，やはり目的格の関係代名詞の省略だとわかります。

-less「できない」で「数えることができない」→「無数の」という意味です。countless timesなら「何度も」ですね。全体では「一度ではなく何度も」の意味です。

### ❹ If 節の意味は？　〔代名詞〕

直訳すると「もし，(あなたが行う)あらゆる選択が，(あなたが)一度ではなく何回も何回も持つことになるであろう結果を運ぶなら」です。一般に，手紙文や対話文以外では，youは「一般の人を示す」働きで，誰か特定の「あなた」に呼びかけているわけではありません。consequenceがマイナスイメージであることを考慮して，日本語を整えれば「自分が行う選択によって，その後，一度ならず何度も大変な目にあうことになってしまうのなら」となります。筆者が仮定法を使ったのはcountless timesに対して「そこまでのことはそう起こらないよな」と思っているからなのだと思われます。

### ❺ decision you はどうなっている？　〔関係詞〕

you would certainly think hardまでは「一生懸命考えるだろう」です。before making a decisionは「決断をする前に」で，makingは動名詞，beforeは前置詞です。

続いてa decision **you**と**名詞の後ろに代名詞が連続**し，その後ろを見ると，**regretに目的語がありません**。よって，youの前に目的格の関係代名詞which[that]が省略されています。訳せば「あとになって後悔するような決断」となります。全体では，「あとになって後悔しないように，何か決断する前によく考える」です。certainlyは文修飾語なので，〈It is certain that S＋V〉と同じように「きっと，～は確実だ」と訳すのが一般的です。

部分訳　あとになって後悔するかもしれないような決断をしてしまう前に，きっと懸命に考えるようになるだろう

## 一言

受験や就職など人生の岐路に立ったときは，失敗しないように必死で考えます。でも，「これが分かれ道だ」と意識していないときの決断がその先大いに影響することもあります(振り返ってみて気づくものですが)。本文では，それが実感として本当にわかっていれば，あらゆる選択や決断の際にもっと考え真剣になるにちがいない，と言っています。

## 解答例

自分が行った選択によって，その後，一度ならず何度も大変な目にあうことになってしまうのなら，あとになって後悔するかもしれないような決断をしてしまう前に，きっとしっかり考えるようになるだろう。
