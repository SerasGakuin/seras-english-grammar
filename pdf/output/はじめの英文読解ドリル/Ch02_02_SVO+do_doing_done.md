# Chapter 2 - 2 >>> SVO＋*do* / *doing* / *done*

## ポイント

SVOCのCに〈*do*〉〈*doing*〉〈*done*〉をとれる動詞がある。
OとCには「主語―述語〔Oが…する/している/される〕」の関係が成立する。

### ① see / hear / feel

| | | |
|---|---|---|
| see | ＋O *do* | Oが…する　のを見る／のが見える |
| hear | ＋O *doing* | Oが…している　のを聞く／のが聞こえる |
| feel | ＋O *done* | Oが…される　のを感じる |
| | ＋O *do* | Oに…してもらう［させる］ |

※「客から店員」,「上司から部下」など, しかるべき相手への依頼・指示

### ② have

| | |
|---|---|
| have | ＋O *done*　Oを…してもらう［される］ |

### ③ make

| | |
|---|---|
| make | ＋O *do*　Oに…させる, SによってOは…する（因果関係） |
| | ＋O *done* |

- make *oneself* understood　（相手に）理解してもらう
- make *oneself* heard　（相手に）声が届く
- make O known　（人に）Oを知らせる

※make O *done*の形は, 上記の定型表現に限る

### ④ let ＋ O *do*

Oに…させる

### ⑤ keep / leave

| | |
|---|---|
| keep | ＋O *doing*　Oが…している |
| leave | ＋O *done*　Oが…される |

状態を保つ／ままにする

※keepは意識的, leaveは無意識的

## ドリル1

文構造を意識して, 例文❶〜❹の灰色の部分をなぞりましょう。

### 例文

**(1) I saw someone enter the building.**

| S | V | O | C |
|---|---|---|---|
| I | saw | someone | enter the building |

- 「誰かが入る」という主述関係が成立。
- see O *do*「Oが…するのを見る／のが見える」

私は, 誰かが建物の中に入るのを見た［入るのが見えた］。

### 練習❶

**(2) I heard a dog barking 〈 in the distance 〉.**

| S | V | O | C | M |
|---|---|---|---|---|
| I | heard | a dog | barking | in the distance |

- 「イヌがほえている」という主述関係が成立。
- hear O *doing*「Oが…しているのを聞く／のが聞こえる」

- bark 動 ほえる

私は, 遠くの方でイヌがほえているのを聞いた［ほえているのが聞こえた］。
