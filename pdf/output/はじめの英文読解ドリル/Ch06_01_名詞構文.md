# Chapter 6 - 1 >>> 名詞構文

## ポイント

「文」でも表現できることを「名詞句」に"圧縮"することがあり，「名詞構文」と呼ぶ。無生物主語構文（p.130）と共存することが多い。

### ❶「元々"動詞"である名詞」を見たら，それを「V'」と捉え，周辺から「S'」や「O'」となりそうな表現を探す。

（例）the arrival of the train（← The train will arrive [arrived].の関係）

V' S'

電車が到着する［した］こと

### ❷「元々"形容詞"である名詞」を見たら，それを「C'」と捉え，周辺から「S'」となりそうな表現を探す。

（例）his sadness（← He is [was] sad.の関係）

S' C'

彼が悲しんでいる［いた］こと

### 主語（S'）の現れ方

| | | |
|:--|:--|:--|
| ①所有格 | S' V' | |
| | his arrival | |
| | 彼が到着する［した］こと | |
| ② of | V' S' | |
| | the arrival of the train | |
| | 電車が到着する［した］こと | |
| ③ by | V' S' | |
| | the education by parents | |
| | 親が教育する［した］こと | |
| ④ on the part of | C' S' | |
| | the rudeness on the part of my son | |
| | 息子が失礼である［であった］こと | |

### 目的語（O'）の現れ方

| | | |
|:--|:--|:--|
| ①所有格 | O' V' | |
| | her protection | |
| | 彼女を保護する［した］こと | |
| ② of | V' O' | |
| | the education of children | |
| | 子どもを教育する［した］こと | |
| ③ その他の前置詞 | | |
| | influence / effect on ～ | |
| | ～に影響を与える［与えた］こと | |
| | control / rule over ～ | |
| | ～を支配する［した］こと | |
| | resemblance / approach to ～ | |
| | ～に似ている［いた］／接近する［した］こと | |

※時制，態は文脈次第である。また，S'とO'がすべてそろうとは限らない。

※上記の訳例では「…こと」としているが，実際には文脈に応じて訳す。

## ドリル1

文構造を意識して，例文にならって，練習❶〜❹の灰色の部分をなぞりましょう。

### 例文

(1) I was surprised〈at his knowledge of the fact〉.

S V C M

名詞構文　He knew the fact.の関係。S V O

私は，彼がその事実を知っていることに驚いた。

### 練習❶

(2)〈On the arrival of the train〉, I heard a strange noise.

M S V O

名詞構文　The train arrived.の関係。S V

電車が到着した時，奇妙な音が聞こえた。

### 練習❷

(3) He lacks concern〈for the influence of TV on teenagers〉.

S V O M

名詞構文　TV influences teenagers.の関係。S V O

lack 動 〜が欠けている

彼は，テレビが10代の若者たちに与える影響への関心が欠けている。

### 練習❸

(4) I am certain〈of his honesty〉.

S V C M

名詞構文　He is honest.の関係。S V C

certain（of ～）形 （〜を）確信して

私は，彼が正直であると確信している。

## ドリル3

文構造を分析し，日本語に訳しましょう。

❶［名詞のカタマリ］（形容詞のカタマリ）〈副詞のカタマリ〉を見つけて，それぞれ［ ］（ ）〈 〉を記しましょう。❷文の要素をⒶ▶に（S・V・O・C・M）で表しましょう。❸名詞構文を探し，Ⓑ▶に（S'・V'・O'・C'）で表しましょう。❹日本語訳▶の空所を埋めましょう。

(1) It was illegal to deny the story of God's creation of man .

日本語訳▶＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿という話を否定することは，違法であった。

illegal 形 違法の／deny 動 〜を否定する／creation 名 創造 > create 動 〜を創造する

(2) The education of children by their parents seems difficult .

日本語訳▶＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿は，難しいようだ。

(3) He is popular with girls because of his resemblance to a popular actor .

日本語訳▶＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿ので，（彼は）女の子に人気だ。

resemblance（to ～）名 （～との）類似 > resemble 動 〜に似ている

(4) The equality of men and women is a fundamental principle .

日本語訳▶＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿は，根本的な原則である。

equality 名 平等 > equal 形 平等である／fundamental 形 根本的な, 基本的な／principle 名 原則

(5) Some people challenge the view of the Japanese as a diligent people .

日本語訳▶一部の人は，＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿に異議を唱えている。

challenge 動 〜に異議を唱える, 〜に疑念を呈する／view 名 見解 > view O as C 動 OをCだと見なす

### 解答

(1) It was illegal［to deny the story of God's creation of man］.

S V C

神が人間を創造したという話を否定することは，違法であった。

(2)［The education of children by their parents］seems difficult.

S V C

親が子どもを教育することは，難しいようだ。

(3) He is popular〈with girls〉〈because of his resemblance to a popular actor〉.

S V C M M

彼はある人気俳優に似ているので，（彼は）女の子に人気だ。

(4)［The equality of men and women］is a fundamental principle.

S V C

男女が平等であること［男女平等］は，根本的な原則である。

(5) Some people challenge［the view of the Japanese as a diligent people］.

S V O

一部の人は，日本人を勤勉な国民だと見なすこと［日本人は勤勉な国民だという見方］に異議を唱えている。

## ドリル4

文構造を分析し，日本語に訳しましょう。

❶［名詞のカタマリ］（形容詞のカタマリ）〈副詞のカタマリ〉を見つけて，それぞれ［ ］（ ）〈 〉を記しましょう。❷文の要素をⒶ▶に（S・V・O・C・M）で表しましょう。❸名詞構文を探し，Ⓑ▶に（S'・V'・O'・C'）で表しましょう。❹日本語訳▶に訳を記しましょう。

(1) We had to change our tactics due to the enemy's destruction of the city .

日本語訳▶＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿

tactics 名 作戦／due to ～ 〜が原因で, 〜のせいで／enemy 名 敵／destruction 名 破壊 > destroy 動 〜を破壊する

(2) I have to apologize for the mistake on the part of my secretary .

日本語訳▶＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿

(3) This novel depicts the age of the king's rule over the country .

日本語訳▶＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿

depict 動 〜を描く／rule（over ～）名 （～に対する）支配 > rule 動 〜を支配する

(4) The teacher didn't notice her assistant's absence .

日本語訳▶＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿

notice 動 〜に気付く／assistant 名 アシスタント／absence 名 いないこと, 欠席, 欠勤 > absent 形 いない, 欠席して, 欠勤して

(5) The application of psychology to education is thought to be effective .

日本語訳▶＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿＿

application 名 応用 > apply O to ～ Oを～に応用する

### 解答

(1) We had to change our tactics〈due to the enemy's destruction of the city〉.

S V O M

敵がその都市を破壊したせいで，私たちは作戦を変えなければならなかった。

(2) I have to apologize〈for the mistake on the part of my secretary〉.

S V M

私の秘書がミスをしたことに対して，私は謝罪しなければならない。

(3) This novel depicts the age〈of the king's rule over the country〉.

S V O M

この小説は，国王がその国を支配していた時代を描いている。

(4) The teacher didn't notice［her assistant's absence］.

S V O

その教師は，彼女のアシスタントがいないことに気付かなかった。

(5)［The application of psychology to education］is thought to be effective.

S V C

心理学を教育に応用することは，効果的であると考えられている。
