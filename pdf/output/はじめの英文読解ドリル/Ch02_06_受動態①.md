# Chapter 2 - 6 >>> 受動態①

## ポイント

受動態を見た時に, 頭の中で"元の形"が瞬時に浮かぶようになろう。

### 受動態の作り方

【Step1】「本来のO」が「S」に移動する ＝ Oが1つ減る（▲）
【Step2】Vを《be動詞＋*done*》の形にする
【Step3】残りは後ろに

「本来のS」は《by＋本来のS》の形で示し, 主に文末に置かれる。訳出は「〜に（よって）」。ただし,「本来のS」は書かれないことが多い。

| 元の形 | 受動態 |
|---|---|
| SVO | O be *done* (S→V) |
| SVO₁O₂ | O₁ be *done* O₂ (S→V) |
| SVO₁O₂ | O₂ be *done* to/for O₁ (S→V) |
| SVO (to be) C | O be *done* (to be) C (S→V) |
| SVO as C | O be *done* as C (S→V) |

- O₁を後ろに残す場合, p.24のタイプAならto, タイプBならforが付く。

## ドリル1

文構造を意識して, 例文にならって, 練習❶〜❹の灰色の部分をなぞりましょう。

### 例文

**(1) My request was ignored 〈 by everyone 〉.**

| S | V | M |
|---|---|---|
| My request | was ignored | by everyone |

- ignore O「Oを無視する」→ O be ignored「Oは無視される」

- request 名 要望
- ignore 動 〜を無視する

私の要望は, 全員によって無視された。

### 練習❶

**(2) We were given sufficient funds ( for the new project ).**

| S | V | O | M |
|---|---|---|---|
| We | were given | sufficient funds | for the new project |

- give O₁O₂「O₁にO₂を与える」→ O₁ be given O₂「O₁はO₂を与えられる」

- fund 名 資金

私たちは, その新しいプロジェクトのための十分な資金を与えられた。
