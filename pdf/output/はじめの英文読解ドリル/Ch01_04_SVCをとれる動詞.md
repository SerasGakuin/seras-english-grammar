# Chapter 1 - 4 >>> SVCをとれる動詞

## ポイント

SVCをとれる動詞は数が限られているので，少しずつ覚えよう。動詞によって，VとCの間にto beが入る場合がある。

### ❶ Cである

- be動詞

### ❷ Cに感じる（五感・印象）

※Cが名詞の場合はV＋like Cとなる。

- look C　Cに見える
- taste C　Cな味がする
- seem *(to be)* C / appear *(to be)* C　Cのようだ，Cに思える
- sound C　Cに聞こえる
- smell C　Cなにおいがする
- feel C　Cに感じる

### ❸ Cのままである（維持・無変化）

- keep C / remain C / stay C　Cのままである

### ❹ Cになる（変化）

※Cに隠れた語がほぼ決まっているものもある。

- prove *(to be)* C / turn out *(to be)* C　Cだと分かる，Cと判明する
- become C / get C　Cになる
- come C　C（良い状態）になる
- grow C　（次第に）Cになる
- turn C　C（主に色）になる
- go C　C（悪い状態）になる
- fall C　（急に）Cになる

## ドリル1

文構造を意識して，例文にならって，練習❶〜❹の灰色の部分をなぞりましょう。

### 例文

(1) You should keep warm〈in winter〉.

S V C M

warmは「名詞を修飾していない形容詞」なので「C」。SVCなので，「You＝warm」が成り立つ。

冬には，あなたは暖かい（状態の）ままでいるべきだ。

### 練習❶

(2) She looks tired〈this morning〉.

S V C M

tiredは「名詞を修飾していない形容詞」なので「C」。SVCなので，「She＝tired」が成り立つ。this / that / last / next / everyに時の名詞が続く場合，前置詞onやinは付かないが，副詞（M）の扱いとなる。

今朝，彼女は疲れているように見える。

### 練習❷

(3) His house looks like a castle.

S V C

look C「Cに見える」のCが「名詞」の場合，《look **like** C》となる。

彼の家は，城のように見える。

### 練習❸

(4) The rumor〈about him〉proved to be false.

S M V C

falseは「名詞を修飾していない形容詞」なので「C」。SVCなので，「The rumor＝false」が成り立つ。

V to be Cのto beは，あってもなくても訳は変わらない。

彼についてのうわさは，間違っていると分かった。

### 練習❹

(5) The infant fell asleep〈in a minute〉.

S V C M

asleepは「名詞を修飾していない形容詞」なので「C」。SVCなので，「The infant＝asleep」が成り立つ。

fall asleepは「眠りにつく」という意味で，ほぼ熟語的に使われる。

その乳児［幼児］は，一瞬で眠りについた。
