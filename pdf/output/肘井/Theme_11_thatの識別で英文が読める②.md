# テーマ11 thatの識別で英文が読める②

## that②の3パターン

| | | |
|---|---|---|
| **1** | so〔such〕とthat | → 因果関係 / 目的・手段の関係 |
| **2** | now that | → 因果関係 |
| **3** | in that | → 因果関係 |

## 例題

次の英文の赤字の単語の役割を意識して、意味を考えなさい。

〈1〉 The rooms are so small that it is hard to wave my arm.
*wave 振る

〈2〉 This was such a hard test that we did not finish it.

〈3〉 He left home early so that he might catch the first train.

〈4〉 Now that he has a child, he has to work harder.

〈5〉 Men differ from animals in that they can think and speak.
*differ 異なる

## 解答・解説

前ページの例題にはすべてthatが使われています。thatとほかの語が一緒になって、接続語をつくっています。soとthatが使われる場合、now that / in thatの表現をおさえてthatを識別し、正しく文を理解しましょう。

### **1** → so〔such〕とthat → 因果関係 / 目的・手段の関係

**英文図解**

〈1〉 The rooms are so small [that it is hard to wave my arm].
　　　S　　　V　　C　　　　　　　M

so～that ...「とても～なので…」ですが、これを覚えただけでは実際の英文では見抜けません。soを見たら、うしろにthatがくると考えることで、so～that ...を必ず見抜けるようになります。thatの手前が原因で、うしろが結果の因果関係をつくります。

和訳　それらの部屋はとても狭いので、自分の腕を振り回すのは難
　　しい。

**英文図解**

〈2〉 This was such a hard test [that we did not finish it].
　　S　　V　　　　C　　　　　　　　M

such～that ...「とても～なので…」です。suchを見たら、うしろにthatがくると考えます。thatの手前が原因で、うしろが結果の因果関係をつくります。soは副詞なのでうしろに形容詞か副詞、suchは形容詞なのでうしろに名詞がきます。

**英文図解**

〈3〉 He left home early [so that he might catch the first train].
　　S　V　　O　　M　　　　　　　M

soとthatが一緒になって使われると、「～するように〔～するために〕」と副詞節をつくり、文の動詞を修飾します。that節のなかに助動詞がくることがほとんどなので、so that S 助動詞の形で覚えます。
so thatの節が「～するように〔～するために〕」という目的で、動詞が手段を表します。この文でも「始発列車に間に合う」という目的のために、「家を早く出る」という手段をとっています。

和訳　彼は始発列車に間に合うように、家を早く出た。

### ポイント21 so thatの識別

| soとthatが離れる | soとthatがくっつく |
|---|---|
| ❶ so～that ... → 例文1 | ❷ so that S 助動詞 → 例文2 |
| 「とても～なので…」 | 「Sが～するように」 |
| | ❸ ～, so that ... → 例文3 |
| | 「～だ。その結果…」 |

soとthatの組み合わせでつくる表現は、上のように3つの可能性があります。まずsoとthatが離れていれば❶で、一緒だと❷か❸のどちらかです。so thatの手前にカンマがあれば❸で、カンマがなければ❷です。

**例文1**　次の文の赤字の単語の役割と意味を考えなさい。
　　This is so beautiful that I can't express it by words.

soとthatが離れているので、「とても～なので…」と因果関係を表します。soを見た時点でthatを予測しましょう。

↓

**英文図解**

This is so beautiful [that I can't express it by words].
S　V　　　C　　　　　　　　M

和訳　これはとても美しいので、言葉では表せない。

**例文2**　次の文の赤字の単語の役割と意味を考えなさい。
　　Let's take an express train so that we can get there
　　on time.

soとthatがくっついて、カンマがないので、「SがVするように」と目的と手段を表す表現です。so that S 助動詞の形です。例文2でも、「私たちが時間通りにそこに着く」という目的のために、「急行列車に乗る」という手段をとっています。

↓

**英文図解**

Let's take an express train [so that we can get there on time].
　V　　　　O　　　　　　　　　　　M

和訳　時間通りにそこに到着できるように、急行列車に乗ろう。

**例文3**　次の文の赤字の単語の役割と意味を考えなさい。
　　He was late for the time, so that he missed the
　　train.

soとthatがくっついて、カンマがあるので、「その結果～」と結果を表します。

↓

**英文図解**

He was late [for the time], [so that he missed the train].
S　V　C　　　M　　　　　　　　M

和訳　彼はその時間に遅れた。その結果、電車を逃した。

### **2** → now that → 因果関係

**英文図解**

〈4〉 [Now that he has a child], he has to work harder.
　　　　　　M　　　　　　　　S　　V　　　　M

thatはnowと一緒になって、副詞節をつくります。「いまや～なので」と、has to workを修飾します。
now thatの節とうしろの文で、因果関係をつくります。この文でも「彼には子どもがいる」という原因で、「もっと一生懸命働かなければならない」という結果を導きます。

和訳　いまや彼には子どもがいるので、もっと一生懸命働かなけれ
　　ばならない。

### **3** → in that → 因果関係

**英文図解**

〈5〉 Men differ [from animals] [in that they can think and
　　S　　V　　　　M　　　　　　　　M
　　speak].

thatはinと一緒になって副詞節をつくります。「～という点で」と、動詞のdifferを修飾します。因果関係を表すこともあり、この文も「人間は考えたり話したりできる」という原因で、「動物と違う」という結果を導くと考えることもできます。

和訳　人間は考えたり話したりできる点で、動物と違う。

## テーマ11 thatの識別で英文が読める② のまとめ

英文中でthat＋αを発見 →so〔such〕～that ... / so that S＋助動詞 / now that / in thatの表現をおさえてthatを識別する。
