# テーマ18 呼応で英文が読める

## 呼応の3パターン

| | | |
|---|---|---|
| **1** | andと呼応する表現 | → both |
| **2** | butと呼応する表現 | → not / not only |
| **3** | or〔nor〕と呼応する表現 | → either / neither |

---
**p.126**
---

## 例題

次の英文の赤字の単語の役割を意識して、意味を考えなさい。

〈1〉 Living in a city has both advantages and disadvantages.
*advantage 利点　　disadvantage 不利な点

〈2〉 The question is not whether we will die, but how we will live.

〈3〉 We need not only good language skills but clear thinking.

〈4〉 You can either fill in this form or email us.
*fill in ～に記入する

〈5〉 They had neither met the author nor read any of his books.
*author 作者

## 解答・解説

前ページの例題はすべて呼応がおきている文です。呼応は前の単語にうしろの単語が引き合う表現です。〈1〉～〈5〉まですべて赤字の単語に反応していては遅いのです。すべてその手前にある単語に反応すると、呼応が見抜けるようになります。

### **1** → andと呼応する表現 → both

**英文図解**

〈1〉 〈Living in a city〉 has both advantages and
　　　　S　　　　　　V　　　　O
　　disadvantages.

「そして」のandではなくて、both *A* and *B*「AとB両方」のandです。bothを見たときに、both *A* and *B* とうしろの形を予測します。

和訳　都市で暮らすことは、利点と不利な点の両方がある。

### **2** → butと呼応する表現 → not / not only

**英文図解**

〈2〉 The question is not 〈whether we will die〉, but 〈how we
　　　S　　　　V　　　　　　　　　　　　　　　C
　　will live〉.

「しかし」を意味するbutではなくて、not *A* but *B*「AではなくてB」のbutです。butに反応すると、「しかし」と思い込んでしまいます。notに反応して、notのうしろにbutがくると構えることで、必ずnot *A* but *B*を見抜くことができます。

和訳　問題は私たちが死ぬかどうかではなくて、どう生きるかだ。

**英文図解**

〈3〉 We need not only good language skills but clear thinking.
　　S　V　　　　　　　　　O

not only *A* but（also）*B*「Aだけでなく Bも」です。not onlyがきたらうしろにbut（also）がくると構えます。alsoはよく消えます。not only *A* but（also）*B* ＝ *B* as well as *A* です。We need clear thinking as well as good language skills.も近い意味です。

和訳　私たちはすぐれた言語技術だけでなく、はっきりとした思考も必要だ。

### **3** → or〔nor〕と呼応する表現 → either / neither

**英文図解**

〈4〉 You can either fill in this form or email us.
　　S　　　V₁　　　O₁　　V₂　O₂

either *A* or *B*「AかBか」です。eitherがきたらうしろにorがくると構えることで、この表現を見抜くことができます。

和訳　あなたはこの申込用紙に記入しても、われわれにＥメールを送ってもよい。

**英文図解**

〈5〉 They had neither met the author nor read any of his
　　S　　V₁　　　　O₁　　　V₂　　　O₂
　　books.

neither *A* nor *B*「AもBもどちらも～ない」です。neitherがきたらうしろにnorがくると構えることで、この表現を見抜くことができます。

和訳　その人たちは、その作者に会ったこともないし、その作者の本を1冊も読んだことがなかった。

## テーマ18 呼応で英文が読める のまとめ

bothを見たらand。notやnot onlyを見たらbut（also）。eitherを見たらor。neitherを見たらnorを予測する。
