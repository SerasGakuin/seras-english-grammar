# OCR目次書き起こし

目次ページを書き起こしてMarkdownファイルとして保存する。

## 引数
- $ARGUMENTS: 書籍名（例: スクランブル、肘井、入門英文）

## 手順

1. **前処理確認**
   - `pdf/images/{書籍名}/` に目次画像があるか確認
   - なければ、PDFから抽出・回転・画像化を実行

2. **目次ページ特定**
   - ユーザーに目次のページ範囲を確認
   - 画像ファイルを特定

3. **書き起こし実行**
   - subagent (general-purpose) に以下を渡す:
     - 画像ファイルパス
     - 表記ルール（`.rules/notation-rules.md`）
     - 出力形式の指示

4. **Codexレビュー**
   - `/ocr-review` を実行
   - 問題があれば修正

5. **保存**
   - `pdf/output/{書籍名}/00_目次.md` に保存
   - `progress/status.json` を更新

## subagentへの指示テンプレート

```
以下の画像ファイルを読み込んで、英語参考書の目次をMarkdown形式で書き起こしてください。

画像ファイル:
{画像ファイルパスのリスト}

表記ルール:
{.rules/notation-rules.md の内容}

ルール:
1. 章番号、タイトル、ページ番号を正確に維持
2. 階層構造を適切にインデントで表現
3. 余計な説明は不要、目次のテキストのみ出力
4. 重複があれば除去して整理

出力形式: Markdownテキストのみ
```

## 注意事項
- オーケストレーター自身は画像を読まない（subagentに委譲）
- 完了後は必ず進捗ファイルを更新
